# Greenhouse-data-monitor
农业大棚数据监控系统
## 文件说明
### index.js 
Web服务器，基于NodeJS，上接Web客户端，下接mysql数据库。监听端口3000</br>
主要功能有：</br>
#### 用户登录
ajax传输前端账号密码，服务器收到后在数据库中查找，如果找到，则登录主界面
#### 数据返回
web前端ajax传输数据请求，服务器从数据库中查到并处理，返回给前端数据。
#### 路径返回
http请求返回相应response
### write_mysql.js
模拟硬件设施部分的汇聚节点向应用服务器发送信息，并将信息发送到数据库的过程</br>
主要功能就是不停的向数据库中写数据，数据是random数据。

登录页面

主页面
